Require

What is the require directive

The #require directive is used to import both macros and real values from a file that has exported them. When requiring macros, one uses their name to refer to them.



How to require

The require directive follows the syntax of #require [value1, value2, value3...] from [path to file]



Example of a full code block 
#require [x, y] from [./export.sly]

console.log($<x + 200> + y);
