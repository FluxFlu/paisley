Compiler Flags

throw-for-errors

The throw-for-errors compiler flag determines whether the program should throw an error on compilation failure, or simply exit cleanly with an exit value of 1. Options are:

  • true
  • false

Default is false.

use-abnormal-filenames

The use-abnormal-filenames compiler flag determines whether filenames that don't end in .sly should be accepted. Options are:

  • true
  • false

Default is false.

keyboard-layout

The keyboard-layout compiler flag determines which keyboard layout you are using, for the purpose of typo analysis. Options are:

  • qwerty
  • azerty
  • dvorak
  • colemac

Default is qwerty.

typo-threshold

The typo-threshold compiler flag determines how far away two keys can be from each other and still be considered a typo. Value can be any integer. Default is 2.

error-threshold

The error-threshold compiler flag determines how many differences two identifiers can have and still be considered similar. Value can be any integer. Default is 3.
