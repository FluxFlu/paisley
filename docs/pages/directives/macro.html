Macro

What is the macro directive

The #macro directive is used to create macros.



How to use

The macro directive follows the syntax of #macro name[params] [code].



What is a macro

A macro is a function that runs at compile time.



Example: 
#macro add[x, y] [x + y]

console.log(add(2, 3));

This will compile to:

console.log(2 + 3);
