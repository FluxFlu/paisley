Export

What is the export directive

The #export directive is used to export both macros and real values from a file so they may be required by other files. When exporting macros, one uses their name to refer to them.



How to export

The export directive follows the syntax of #export [value1, value2, value3...]



Example of a full code block 
#define x [200]

const y = x + 200;

#export [x, y]
